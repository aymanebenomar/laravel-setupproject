# Learning Streak Tracker 🎯

A fast, scalable React app for tracking learning streaks publicly. Includes authentication, goal creation, daily check-ins, and public profiles.

## Features

- **Authentication**: Secure login and registration
- **Goal Creation**: Create and manage learning goals
- **Daily Check-ins**: Track your progress with daily updates
- **Public Profiles**: Share your learning journey with others
- **Performance Optimized**: Built with React + Vite for lightning-fast performance
- **Modern UI**: Tailwind CSS 3.3.3 for beautiful, responsive design

## Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS 3.3.3
- **Node.js**: v18+

## Getting Started

### Prerequisites

- Node.js v18 or higher
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd learning-streak-tracker
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

The app will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
